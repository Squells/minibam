# Установка модулей

Сделайте следующие (способ через файл):

1. Скачайте архив "modules.zip"(изм. 27.01.23) из репозитория
2. Содержимое архива загрузите полностью в главную директорию вашего сайта
3. Перейти по ссылке: [ваш сайт]/minibam.php, после установки перейдите к установке модулей, по адресу: [ваш сайт]/admin/minibam
